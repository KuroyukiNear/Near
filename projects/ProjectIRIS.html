Project IRIS

Overview | Changelog

Overview

A Discord bot started in October 2021. Originally intended to be a multi purpose bot. Then used as a economy bot, multi purpose bot, music bot, and a logging bot. Currently being used as a logging bot for a specific private server to log user activities. The control center is currently located in Near's private server.
