Overview

Eyes in Every Corner, Whispers in Every Ear.
The Overseer Project - Where Every Move Echoes in the Shadows of Surveillance.

A Discord bot made to monitor every single activity in a specific Discord server made for this very purpose. A social experiment started by Near, inspired by the NoCom Exploit by the Nerds inc. on 2b2t. This bot will log every single user activity into a database for further analysis. Various user activities are being logged and monitored, including: messages sent, messages edited, messages deleted, voice events, user joins and leaves, and user profile changes. It also scraps all user data that is public, including: username, user ID, date the user joined Discord and their About Me.
